# UI Автотесты на Playwright для SauceDemo

Этот репозиторий содержит набор базовых UI-автотестов для учебного сайта [SauceDemo](https://saucedemo.com). Проект создан для демонстрации навыков работы с Playwright, JavaScript/TypeScript и основами построения тестовых сценариев.

## 🚀 Стек технологий

- **Framework:** Playwright
- **Language:** JavaScript/TypeScript

## 📋 Какие проверки реализованы:

1. **Авторизация:** успешный вход в систему с валидными данными.
2. **Корзина:** добавление нескольких товаров и проверка счетчика на иконке корзины.
3. **Сортировка:** проверка корректности сортировки товаров по имени (A-Z).
4. **Разлогин:** выход из системы через боковое меню.

## 🛠 Установка и запуск

1. **Клонируйте репозиторий:**

   ```bash
   git clone https://github.com/MaximDudkin/saucedemo-playwright-tests.git
   ```

2. **Установите зависимости:**

   ```bash
   npm install
   ```

3. **Запустите тесты:**

   ```bash
   npx playwright test
   ```

4. **Запуск в UI-режиме (с визуализацией):**
   ```bash
   npx playwright test --ui
   ```
